Detailed Description

The Texture Loader makes loading in textures far more efficient as a new texture doesn't need to be loaded in everytime a new instance of something is created.

All textures need to be loaded in once into the textures vector and then they can all be accessed by any class thats needs them by getting the texture Iterator

Usage example:

void Car::setCarTexture(vector<Texture>::iterator GivenTexture)
{
m_render.setTexture(*GivenTexture);
}

Detailed Description

HUD (Heads Up Display) is a class which manages all the infomation the player needs to know in order to play the game. It creates the RPM counter, gear counter and lap and best time counter.

HUD also ensures the gear values and needle position are correct and to the actions of the car.

HUD also manages the checkpoints for the lap times. The checkpoints are float rects which can be collided with in the Game class which would then in turn give the infomation to update the timer or best time

Usage example:

HUD();

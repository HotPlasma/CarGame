Detailed Description

Game is absolutely essential to the game. It pulls together all of the other classes and makes them function with eachother in order to maken this game playable.

Game is in charge of assigning textures and drawing everything to the screen and deciding on when a player is accelerating or breaking. It also ensures that all the gears and RPM values change in order to create realistic driving and movement.

Game proccesses all the keyboard controls and makes sure the game responds to player input. On top of this Game ensures collisions are checked by everything and tells what functions need to be run in order to resolve the collisions.

Game is in charge of loading in the map from the texture loader for the player to race on.

Game also handles I/O in the form of reading tyre positions from a file and creating and drawing them to the screen.

Detailed Description

Tyre is class which holds all infomation about the m_Tyres in the game. To be instantiated it needs to be given a position and inverse mass.

Tyre is also a collidable, which means that it can collide with any other collidable. It is able to collide with other m_Tyres using the circle-circle collison method.

Usage example:

Tyre(myVector(50,40), 0.3))

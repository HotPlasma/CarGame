\subsection{Detailed Description}
\hyperlink{class_game}{Game} is absolutely essential to the game. It pulls together all of the other classes and makes them function with eachother in order to maken this game playable.

\hyperlink{class_game}{Game} is in charge of assigning textures and drawing everything to the screen and deciding on when a player is accelerating or breaking. It also ensures that all the gears and R\+P\+M values change in order to create realistic driving and movement.

\hyperlink{class_game}{Game} proccesses all the keyboard controls and makes sure the game responds to player input. On top of this \hyperlink{class_game}{Game} ensures collisions are checked by everything and tells what functions need to be run in order to resolve the collisions.

\hyperlink{class_game}{Game} is in charge of loading in the map from the texture loader for the player to race on.

\hyperlink{class_game}{Game} also handles I/\+O in the form of reading tyre positions from a file and creating and drawing them to the screen.
